# Referencias Adicionales y Librerías
*Auditoría del Stack Tecnológico - 2026-01-26*

## 1. Librerías Clave (Requirements)
Referencias de documentación oficial para las dependencias principales.

### Backend Framework
- **FastAPI** (v0.128): [fastapi.tiangolo.com](https://fastapi.tiangolo.com)
- **SQLAlchemy** (v2.0): [docs.sqlalchemy.org](https://docs.sqlalchemy.org)
- **Alembic**: Migraciones de base de datos.
- **Pydantic Settings**: Gestión de configuración.

### AI & Procesamiento
- **Azure Speech SDK** (v1.34+): [learn.microsoft.com/azure/ai-services/speech-service](https://learn.microsoft.com/en-us/azure/ai-services/speech-service/)
    - STT (Reconocimiento) y TTS (Síntesis Neural).
- **Groq Python SDK**: Llama 3 inference ultrarrápida.
- **Silero VAD** (OnnxRuntime): Detección de actividad de voz local.

### Telefonía
- **Telnyx Call Control v2**: [developers.telnyx.com](https://developers.telnyx.com/docs/v2/call-control)
- **Twilio Media Streams**: [twilio.com/docs/voice/media-streams](https://www.twilio.com/docs/voice/media-streams)

### Infraestructrura
- **Redis**: [redis.io](https://redis.io) - Gestión de estado y caché.
- **Prometheus**: Monitoreo y métricas.
